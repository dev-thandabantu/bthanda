# bthanda.vercel.app

Personal site for Brighton Tandabantu — AI engineer, founder, and builder.

## What's here

- **Portfolio** — AnchorBase, AgriData AI, AakiTech, BOQ Generator, and other projects
- **CV** — downloadable as PDF and DOCX, generated from a single source of truth in `src/lib/cvData.ts`
- **Family tree** — six documented attempts at rendering a polygamous family graph that breaks every standard genealogy tool. 218 relatives across Zimbabwe, South Africa, and Mozambique. Both grandfathers had multiple wives. The data model is a directed graph, not a binary tree.
- **Job application tracker** — private, at `/applied`

## Stack

Next.js 15 · TypeScript · Tailwind CSS · Vercel

## Running locally

```bash
npm install
npm run dev
```
